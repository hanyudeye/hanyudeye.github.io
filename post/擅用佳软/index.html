查找工具

计算机中的资源，是以目录形式分布在目录树中的，一个好用的查找工具可以迅速定位到要修改的资源。好用的工具有

  • Everything: Windows下强大的文件搜索工具
  • Listary: Windows 下的文件搜索和应用启动工具
  • ulauncher: Linux 下具有同Listary 相似功能的工具
