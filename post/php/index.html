单元测试 phpunit

功能选项

  • 显示 PHP information and configuration ( -i )

  • 显示配置文件 (–ini)

  • 显示编译好的模块 (-m)

  • 打开自带服务器监听端口 (-S addr:port)

  • 显示一些信息 ( –rf 函数，–rc 类名 ,–re 扩展名)

Apache2 选项

  • 对配置文件进行语法测试 (-t)

  • 测试后另外输出虚拟主机信息 (-t -D DUMP_VHOSTS )

  • 测试后另外输出加载的模块 (-t -D DUMP_MODULES)

2

  # 例子
  apachectl -t -D DUMP_MODULES
